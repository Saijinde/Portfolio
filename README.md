# Apple-Style Interactive Resume

A premium, scroll-based interactive resume website inspired by Apple's product launch pages.

## Features
- **Minimalist Aesthetic**: Clean dark mode design with high-quality typography (Inter).
- **Interactive Scroll**: Text reveal animations and section fades using GSAP and ScrollTrigger.
- **Project Showcases**: High-impact visuals with parallax effects.
- **Responsive**: Fully optimized for desktop and mobile devices.
- **Apple Design Patterns**: Large whitespace, large typography, and smooth easing.

## Tech Stack
- HTML5 / CSS3
- JavaScript
- [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
- [ScrollTrigger](https://greensock.com/scrolltrigger/)
- [ScrollToPlugin](https://greensock.com/scrolltoplugin/)

## Setup
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. For the best experience, use a local server (e.g., Live Server in VS Code).
